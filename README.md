# FinalSysDevProject

Wampoon-ready Slim PHP inventory system.

## How to run
1. Put the folder in your Wampoon/WAMP `www` folder.
2. Import `database/projectdb_final.sql` into phpMyAdmin.
3. Make sure the database is named `projectdb`.
4. Open: `http://localhost/FinalSysDevProject/`

## Logins
- Owner/Admin: `admin` / `admin123`
- Employee: `employee` / `employee123`

## Important
This version works like the lab project: the Slim front controller is at the project root (`index.php`), and `.htaccess` redirects routes to it.
